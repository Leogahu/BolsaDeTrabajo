# ChapaTuChamba - Plataforma de Primer Empleo

## 📌 Visión General

**ChapaTuChamba** es una plataforma web diseñada para conectar a jóvenes de 18 a 25 años sin experiencia laboral con empresas que ofrecen prácticas, trainee o primer empleo real. A diferencia de LinkedIn o Computrabajo, aquí no se piden años de experiencia para puestos de entrada.

## 🎯 Problemática que Resuelve

- El 80% de ofertas "junior" o "trainee" exigen experiencia previa (creando un círculo vicioso)
- Los jóvenes envían 10-30 postulaciones al mes sin respuesta
- Los reclutadores no tienen herramientas para evaluar potencial vs trayectoria

## 👥 Segmentos Objetivo

| Segmento | Edad | Características |
|----------|------|----------------|
| Jóvenes sin experiencia | 18-25 años | Estudiantes últimos ciclos, egresados, sin experiencia formal |
| Reclutadores / PYMES | - | Buscan talento joven con habilidades blandas y técnicas |

## 🔍 Hallazgos Clave (Entrevistas)

- Frustración por requisitos contradictorios
- Necesidad de **feedback específico** tras rechazo
- Deseo de validar habilidades mediante **pruebas prácticas**
- Las **recomendaciones académicas** tienen alto valor percibido
